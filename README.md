# CTF-Write-ups

Write-ups for CTF challenges.

# Format

## Directory Structure

```
- CTF Name
  - README.md
  - Category
    - Challenge Name
     - static (for images)
     - binaries, etc
     - README.md (solution)
```

## Challenge Solution

```
# Challenge Name

Author: [author](https://github.com/author)

Brief Description

# Requirements

- List of requirements

# Source

- Links to source code and other files (if applicable).

# Exploitation

Detailed Exploit with images (if any).
```
